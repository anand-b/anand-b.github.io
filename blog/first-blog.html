First blog post This is my very first blog post after creating this repository. This blog describes how every other blog post will be created.
  • Clone the repository.
  • In the <REPO_HOME>/content/blog directory, create a new blog post with the file name same as the one that will appear in the URL.
  • The post must be enclosed within <blog-post></blog-post> tag.
  • The title must be enclose in <blog-title></blog-title> tag.
  • The content must be enclosed in <blog-content></blog-content> tag.
  • Go to <REPO_HOME>. Run the build.py python script with <REPO_HOME> as the current working directory.
  • Push your changes in the repository.
